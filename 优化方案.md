### 2.2 文件完整性检查缺失

**问题**：
- 文件还在写入时就被处理
- 导致 Eagle 导入不完整或失败

**优化方案**：
```python
def _wait_for_file(self, file_path: str, timeout: int = 30):
    """等待文件写入完成"""
    import time
    start = time.time()
    last_size = -1

    while time.time() - start < timeout:
        try:
            current_size = os.path.getsize(file_path)
            if current_size == last_size and current_size > 0:
                return True  # 文件大小稳定，写入完成
            last_size = current_size
        except OSError:
            pass
        time.sleep(0.5)

    return False  # 超时
```

### 2.3 决策引擎误匹配

**问题**：
- `is_vague_name()` 判断逻辑过于简单
- 关键词匹配可能误匹配（如"白"会匹配到"白起"）
- 没有考虑文件扩展名（非图片文件也被处理）

**优化方案**：
```python
# 改进 is_vague_name
def is_vague_name(filename: str) -> bool:
    stem = Path(filename).stem
    # 纯数字
    if re.match(r"^\d+$", stem):
        return True
    # 纯字母无意义串（至少5个字符）
    if re.match(r"^[a-zA-Z]{5,}$", stem) and not re.match(r"[aeiou]{3}", stem, re.I):
        return True
    # 哈希值
    if re.match(r"^[a-f0-9]{32}$", stem, re.I):
        return True
    return False

# 改进关键词匹配
def match_by_filename(filename: str) -> Optional[dict]:
    stem = Path(filename).stem.lower()
    # 使用完整词匹配，避免部分匹配
    for keyword, info in mapping.items():
        if re.search(rf'\b{re.escape(keyword)}\b', stem):
            return info
```

### 2.4 AI 分析可靠性问题

**问题**：
- Qwen-VL 调用没有重试机制
- 响应解析不够健壮
- 没有缓存机制（相同图片重复分析）

**优化方案**：
```python
# 添加重试机制
def analyze_image(file_path: str, max_retries: int = 3) -> Optional[dict]:
    for attempt in range(max_retries):
        try:
            result = _call_qwen_vl(file_path)
            if result:
                return result
        except Exception as e:
            if attempt < max_retries - 1:
                time.sleep(2 ** attempt)  # 指数退避
            else:
                _LOG.error(f"AI 分析失败：{e}")
    return None

# 添加缓存
import hashlib

def _get_image_hash(file_path: str) -> str:
    with open(file_path, "rb") as f:
        return hashlib.md5(f.read()).hexdigest()

def analyze_image_with_cache(file_path: str) -> Optional[dict]:
    cache_path = DATA_DIR / "cache" / f"{_get_image_hash(file_path)}.json"
    if cache_path.exists():
        return json.loads(cache_path.read_text())

    result = analyze_image(file_path)
    if result:
        cache_path.parent.mkdir(parents=True, exist_ok=True)
        cache_path.write_text(json.dumps(result, ensure_ascii=False))
    return result
```

### 2.5 知识库学习机制问题

**问题**：
- 记录匹配时没有考虑用户反馈
- 没有清理过期或低置信度条目的机制
- 来源匹配逻辑过于简单

**优化方案**：
```python
# 添加用户反馈机制
def record_feedback(keyword: str, theme: str, is_correct: bool):
    data = _load()
    mapping = data.setdefault("keywords_mapping", {})

    if keyword in mapping:
        entry = mapping[keyword]
        if is_correct:
            entry["positive_feedback"] = entry.get("positive_feedback", 0) + 1
        else:
            entry["negative_feedback"] = entry.get("negative_feedback", 0) + 1
            # 降低置信度
            entry["confidence"] = max(0.3, entry["confidence"] - 0.1)
    _save(data)

# 添加清理机制
def cleanup_stale_entries(max_age_days: int = 90):
    data = _load()
    mapping = data.get("keywords_mapping", {})
    cutoff = datetime.now() - timedelta(days=max_age_days)

    to_delete = []
    for keyword, info in mapping.items():
        first_seen = info.get("first_seen", "")
        if first_seen and datetime.fromisoformat(first_seen) < cutoff:
            if info.get("confidence", 0) < 0.5:
                to_delete.append(keyword)

    for keyword in to_delete:
        del mapping[keyword]

    _save(data)
```

### 2.7 配置管理问题

**问题**：

- 没有配置热重载

**优化方案**：

# 添加配置验证
def validate_config(cfg: dict) -> list[str]:
    errors = []

    if not cfg.get("eagle", {}).get("host"):
        errors.append("缺少 eagle.host 配置")

    if not cfg.get("paths", {}).get("downloads"):
        errors.append("缺少 paths.downloads 配置")

    downloads = cfg["paths"]["downloads"]
    if not os.path.isdir(downloads):
        errors.append(f"下载目录不存在：{downloads}")

    return errors

# 添加配置热重载
import signal

def _reload_config(signum, frame):

    _LOG.info("配置已重载")

signal.signal(signal.SIGUSR1, _reload_config)

### 3.2 线程安全问题

**问题**：
- `watcher.py` 在后台线程运行，但访问共享状态
- `menu_app.py` 在主线程，但调用阻塞操作
- 没有使用锁保护共享资源

**优化方案**：
```python
import threading

class SharedState:
    def __init__(self):
        self._lock = threading.Lock()
        self._current_theme = None
        self._known_files = set()

    @property
    def current_theme(self):
        with self._lock:
            return self._current_theme

    @current_theme.setter
    def current_theme(self, value):
        with self._lock:
            self._current_theme = value
```

### 3.3 异常处理不完善

**问题**：
- 很多地方使用 bare `except Exception`
- 没有区分可恢复和不可恢复错误
- 没有错误上报机制

**优化方案**：
```python
# 定义具体异常类型
class EagleConnectionError(Exception):
    pass

class ThemeNotFoundError(Exception):
    pass

class AIAnalysisError(Exception):
    pass

# 分层处理异常
try:
    result = eagle.add_from_path(file_path)
except EagleConnectionError as e:
    _LOG.error(f"Eagle 连接失败：{e}")
    # 重试或等待
except Exception as e:
    _LOG.error(f"未知错误：{e}", exc_info=True)
    # 上报错误
```
