首次使用配置

素材管家需要连接 Eagle 才能自动导入素材。

  1. 打开 Eagle → 设置 → 开发者选项
  2. 勾选「允许其他应用连接」并复制 API Token
  3. 粘贴到下方输入框，点击「保存并连接」
