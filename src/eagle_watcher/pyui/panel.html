⚠️ 首次使用配置

素材管家需要连接 Eagle 才能自动导入素材。请按以下步骤配置：

  1. 打开 Eagle 应用
  2. 进入 Eagle → 偏好设置 → 插件
  3. 勾选「允许其他应用连接」
  4. 复制显示的 API Token
  5. 粘贴到 ~/.eagle-watcher/config.yaml 的 eagle.token 字段
  6. 保存后点击下方「重新检测」
⚠️
文件夹权限不足
监控目录 无访问权限，素材将不会被自动检测。
